# M Official Guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定域名或具体网站，主要用于资料整理与展示。

## 项目简介

本项目旨在以静态 HTML 文件的形式，集中存放并管理一系列独立页面。每个页面均可单独访问，适合用于文档说明、信息展示或轻量级内容发布。

## 目录说明

```
m-official-guide/
├── index.html          # 仓库首页（如有）
├── pages/              # 存放主要 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 资源文件（CSS、JS、图片等）
```

- `pages/`：包含所有独立 HTML 页面，文件名通常反映页面内容主题。
- `assets/`：存放页面引用的静态资源，如样式表、脚本文件或图像。

## 页面归档说明

本仓库中的 HTML 页面均为独立文件，不依赖外部数据库或后端服务。每个页面可直接在浏览器中打开运行，便于离线查看或部署到任意静态托管平台。

页面内容可能涉及不同主题，但均不指向任何具体网站或商业服务。如有更新，将直接替换或新增对应文件。
